# `/frontend-design` -- Distinctive Frontend Design

Create distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics, using a library of curated design examples as grounded inspiration.

## Usage

```
/frontend-design
```

Or just ask Claude to design, build, style, or beautify a UI, page, component, or landing page.

## Features

- A library of 9 curated HTML design examples (8 single-page + one multi-page SaaS site) spanning neo-brutalist, dark-hacker, cinematic SaaS, Swiss-minimal, industrial, and editorial-luxury aesthetics
- A condensed example catalog with real CSS snippets, plus full source files to internalize complete design systems before building
- Match / derive / blend workflows to adapt examples to any brand or content
- Aesthetic guidelines for typography, color, motion, texture, and spatial composition that steer away from generic AI patterns
- Phase 2: extract a chosen prototype into a reusable design system (`tokens.css`, `components.css`, `DESIGN.md`)

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r frontend-design ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse frontend-design "$env:USERPROFILE\.claude\skills\"
```
