# `/project-guidelines-example` -- Project-Specific Skill Template

A worked example of a project-specific skill you can copy as a template for documenting your own project's conventions, architecture, and workflows.

## Usage

Use this as a reference/template. Copy it into a project and adapt the sections to your own stack, structure, and guidelines.

## Features

- A complete, realistic example of a project-guidelines skill (modeled on a real production application)
- Sections for architecture, tech stack, conventions, common tasks, and environment setup
- Demonstrates how to structure project-specific context so Claude Code follows your project's rules
- A starting point for authoring your own project-guidelines skill

## Installation

Copy to your Claude Code skills directory:

**Linux/macOS:**
```bash
cp -r project-guidelines-example ~/.claude/skills/
```

**Windows PowerShell:**
```powershell
Copy-Item -Recurse project-guidelines-example "$env:USERPROFILE\.claude\skills\"
```
